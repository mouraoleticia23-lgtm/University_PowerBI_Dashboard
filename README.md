# University_PowerBI_Dashboard
Dashboard designed to analyze and visualize university academic data. This project provides actionable insights into student demographics, graduation trends, and departmental performance to help university administrators make data-driven decisions.

Key Features & Insights: • Student Overview: Tracking total student count, gender distribution, and enrollment status • Academic Performance: Analysis of average GPAs across different colleges (Engineering, Science, Business, Arts), • Graduation Trends: Visualizing graduation rates over time to identify growth or decline, • Departmental Deep-Dive: Interactive filters to compare performance metrics between various departments • Demographic Analysis: Insights into the student body composition by gender and college.

Technologies used:
• Power BI for data modeling and visualization.
• DAX: used for calculating complex metrics like Average GPA and Graduation Rate %.
• Power Query: For data cleaning and transformation (ETL).
• Dataset: University academic records (anonymized).
